Ruby and Rails 入门学习资料推荐

1. Ruby基础教程

关于Ruby的基础教程，我推荐一本pdf教程《PROGRAMMING RUBY》。这本书讲解了Ruby的基础语法，带领从没有接触过Ruby语言的初学者入门。

2. Ruby on Rails

  • 对于Rails框架教程，首先我要推荐的是張文鈿编写的《Ruby on Rails 實戰聖經》。这本教程主要针对有Ruby入门基础，但没有接触过Rails框架的初学者。这个教程是网页形式，你可以step by step地学习，通过学习此教程，你可以编写简单的rails网站。
  • 其次我推荐一本经典的Rails入门书籍《Web开发敏捷之道——应用Rails进行敏捷Web开发》。这本书也是针对与初学者的，但是它照比《实战圣经》要详细一些。我推荐你先照着张文细的实战圣经的操作一遍，大概有个感性的了解，然后再详细的看这本书，可以解决圣经中不懂的地方。

end

Blog框架汇总

这里汇总了几款静态Blog框架，大家可以酌情选择。

1. Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

2. Jekyll

Jekyll 是一个简单的博客形态的静态站点生产机器。它有一个模版目录，其中包含原始文本格式的文档，通过 Markdown （或者 Textile） 以及 Liquid
转化成一个完整的可发布的静态网站，你可以发布在任何你喜爱的服务器上。Jekyll 也可以运行在 GitHub Page 上，也就是说，你可以使用 GitHub 的服务来搭建你的项目页面、博客或者网站，而且是完全免费的。

3. Octopress

octopress是基于Jekyll开发而成的博客框架。使用Octopress博客最大的优势就是静态化，不依赖脚本程序，没有MysqL等数据库，因此它可以在一些性能差的服务器或者虚拟空间上运行，在同等条件下打开页面的速度自然也是非常快。
Octopress支持自定义模板，可以markdown，文本文件写作、保存可以直接用dropbox等网盘来同步。另外它还可以让你有git版本控制功能。octopress还有个优点就是它的整个使用/发布是个松耦合的结构，相信一些习惯了代码的朋友很喜欢。

end
