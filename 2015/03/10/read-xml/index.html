ios解析xml文件

  • ios
  • os x

ios自带解析xml类NSXMLParser。
自身类必须实现NSXMLParserDelegate协议
首先定义NSXMLParser类

let smlFile = NSXMLParser(data: smlData)
smlFile.delegate = self//设置
smlFile.parse()

然后实现几个协议方法

func parser(parser: NSXMLParser!, didStartElement elementName: String!, namespaceURI: String!, qualifiedName qName: String!, attributes attributeDict: [NSObject : AnyObject]!)//开始解析元素

func parser(parser: NSXMLParser!, foundCharacters string: String!)//解析内容

func parserDidEndDocument(parser: NSXMLParser!)//解析结束
